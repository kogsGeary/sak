sak

Speecher Assistive Kit.

For fpc, fpGui, Lazarus, Delphi On Linux, Windows, Mac, Android,...

With sak, your application become assistive directly, without changing anything in your code.

Multi languages, multi voices, multi forms.

You do not blow up your application, only 1 megas with multi-languages-voices and libraries. You do not need external system like ST-SPI, Orca, SAPI nor external libraries ... Sak uses runtime dynamic loading of libraries so you can choose where to install the kit.

Sak uses the fabulous eSpeak and PortAudio Open Source Libraries.
